# save file bot
Позволяет работать с файловой системой, но по задумке предполагалось, что он будет облачным хранилищем на основе серверов телеграмма.
Бот может хранить иерархию директорий и подсказывать с названием новой папки при её создании и позволяет манипулировать папками, однако есть ограничения в виде уникальности названия каждой папки, это было сделано для быстрого поиска в дальнейшем.

While it allows for interaction with the file system, the original concept envisioned it as a cloud storage solution powered by Telegram's servers. The bot is capable of maintaining a directory hierarchy—even suggesting names when creating new folders—and enables folder manipulation; however, it imposes a restriction requiring that every folder name be unique—a design choice implemented to facilitate rapid searching in the future.
